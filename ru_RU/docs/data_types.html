Интерфейс GormDataTypeInterface

GORM будет читать тип столбца базы данных из tag type, если не найдено, проверит, реализован ли интерфейс struct GormDBDataTypeInterface или GormDataTypeInterface и будет использовать его результат в качестве типа данных

Результат GormDataType будет использован в качестве общего типа данных и может быть получен из schema.Field DataType поля, что может быть полезно при написании плагинов или хуков , например:
